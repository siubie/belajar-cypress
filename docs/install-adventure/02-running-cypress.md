# Membuka Cypress

- Jalankan dengan npx
  ```bash
  npx cypres open
  ```
- Jalankan dengan yarn
  ```bash
    yarn run cypress open
  ```
- Jalankan pake node modules path
  ```bash
  ./node_modules/.bin/cypress open
  ```
